Modern JavaScript Features You Should Know in 2024

By Miguel December 15, 2024
AI Machine Learning Neural Networks

Optional Chaining (?.)

Optional chaining allows you to safely access deeply nested object properties without having to validate each reference in the chain explicitly. 

// Before optional chaining
if (user && user.address && user.address.street) {
  console.log(user.address.street);
}

// With optional chaining
console.log(user?.address?.street);

Nullish Coalescing (??)

The nullish coalescing operator provides a way to fall back to a default value when dealing with null or undefined, but not other falsy values like 0 or empty strings. 

// Using || operator (problematic with falsy values)
const config = {
  timeout: 0,
  retries: null
};

const timeout = config.timeout || 5000; // 5000 (wrong!)
const retries = config.retries ?? 3; // 3 (correct!)

Top-Level Await

You can now use await at the top level of modules, making it easier to work with asynchronous code during module initialization. 

// In a module file
const data = await fetch('/api/config').then(r => r.json());
const connection = await connectToDatabase();

export { data, connection };

Array Methods: at(), findLast(), and findLastIndex()

New array methods provide more intuitive ways to access and find elements. 

const arr = [1, 2, 3, 4, 5];

// Negative indexing with at()
console.log(arr.at(-1)); // 5
console.log(arr.at(-2)); // 4

// Finding from the end
const users = [
  { id: 1, active: false },
  { id: 2, active: true },
  { id: 3, active: true }
];

const lastActiveUser = users.findLast(user => user.active);
console.log(lastActiveUser); // { id: 3, active: true }

Private Fields and Methods

True privacy in JavaScript classes using the # prefix for private fields and methods. 

class BankAccount {
  #balance = 0;
  #accountNumber;

  constructor(accountNumber) {
    this.#accountNumber = accountNumber;
  }

  #validateAmount(amount) {
    return amount > 0 && typeof amount === 'number';
  }

  deposit(amount) {
    if (this.#validateAmount(amount)) {
      this.#balance += amount;
    }
  }

  getBalance() {
    return this.#balance;
  }
}

Logical Assignment Operators

Combine logical operators with assignment for more concise code. 

// Logical OR assignment (||=)
user.name ||= 'Anonymous';

// Logical AND assignment (&&=)
user.isAdmin &&= user.permissions.includes('admin');

// Logical nullish assignment (??=)
config.timeout ??= 5000;

Conclusion

These modern JavaScript features significantly improve code readability, safety, and developer experience. As browser support continues to improve, incorporating these features into your development workflow will make your code more robust and maintainable.
